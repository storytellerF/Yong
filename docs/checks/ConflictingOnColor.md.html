For example, a button colored `primary` will have `onPrimary` text. Because of this, it is important that there is only one possible `onColor` for a given color value, otherwise there is no way to know which 'on' color should be used inside a component. To fix this either use the same 'on' color for identical background colors, or use a different background color for each 'on' color.
