As a result, each factory function *must* use the receiver `Modifier` parameter, to ensure that the function is returning a chain that includes previous items in the chain. Make sure the returned chain either explicitly includes `this`, such as `return this.then(MyModifier)` or implicitly by returning a chain that starts with an implicit call to another factory function, such as `return myModifier()`, where `myModifier` is defined as `fun Modifier.myModifier(): Modifier`.
